# WideDeepLearning

Provides simple interface for Wide and Deep Learning model with TensorFlow.

# Requirements

- Python>=3.5
- TensorFlow>=1.4


# Installation

```sh
$ git clone git@github.com:nukui-s/WideDeepLearning.git
$ cd WideDeepLearning
$ python setup.py install
```
